ModdersToolkit is a toolkit for modders, hence the name.

Current Tools
-C# REPL, exaluate and run code interactively (on server console as well)
-Hitbox visualizer
-Dust tool
-Item defaults tool
-Projectile spawner
-Player layer viewer
-Interface layer viewer.
-Texture updating/editing tool

Planned Tools
-WorldGen tools
-Sound explorer/tester
-NPC Loot validator
-NPC Spawn validator
-NPC kill visualizer
